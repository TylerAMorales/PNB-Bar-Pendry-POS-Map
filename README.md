# Bar POS Terminal Heat Map

## Overview
This project creates a **visual heat map of POS terminal locations** within the bar using configuration data from six Oracle Micros terminals. The purpose of the heat map is to provide a clear visualization of where each terminal is physically located and how they are distributed across the bar floor plan.

The visualization helps with **system documentation, troubleshooting, and operational awareness** by showing the relationship between POS terminals and the bar layout.

## What the Project Does
- Displays a **bar floor plan layout**
- Plots the **locations of six Micros POS terminals**
- Associates each terminal with its **configuration information**
- Provides a **visual heat map representation** to make terminal placement easy to understand
- Helps staff and IT quickly identify terminal positions for maintenance or troubleshooting

## Terminal Information Included
For each POS terminal, the visualization includes:
- Terminal ID / Name
- Location within the bar
- Configuration details
- Position mapped on the floor plan

## Use Case
This tool is primarily used for:
- POS infrastructure documentation
- IT troubleshooting
- Network planning
- Operational reference for staff

## Notes
The heat map is based on the bar’s floor plan and the configuration data of the six Micros POS terminals currently deployed in the environment.
